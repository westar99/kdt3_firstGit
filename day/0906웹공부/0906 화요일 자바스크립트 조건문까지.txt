   ---HTML---
   뭐의 약자지? 
		HyperText Markup
   왜쓰지?  
		웹에서 형식과 구조라는 메타데이터(정보를 위한 정보)를 포함한 '정보'를 전달하기 위해. 
   html문서 만들려면 어떻게 해야되지?  
		확장자를 .html파일로 만들고 제일 위에 <!DOCTYPE html>
   중요한 태그 뭐있지?
		head, body, a, p, div, span, img, table, tr, th, td
		br, h1~h6, link, style, script
		ul, ol, li
		form, input
   그 태그들은 뭐의 약자지?
		a : anchor
		div : division
		img : image
		tr : table row
		th : table head
		td : table data
		br : break row
		h : headline, headlight
		ul : unordered list
		ol : ordered list
		li : list item
   HTML에서 css어떻게 호출하지?
		- <태그 style="속성 : 값">   태그의 인라인 속성으로 css적용 할수있다.
		- <style>태그를 만나면 </style>태그를 만날때까지 html문법이 아닌 css문법으로 해석한다.
		- <link rel="stylesheet" href="css파일주소" >  html외부에서 css파일을 참조한다.
   HTML에서 JS어떻게 호출하지?
		- <script> 태그를 만나면 </script>를 만날때까지 html문법이 아닌 js문법으로 해석한다.
		- <script src="자바스크립트파일주소"></script>  html외부에서 js파일 참조한다.
   시멘틱태그가 뭐지?
		Sementic. 의미론적인. 
		HTML의 본질인 '정보'전달을 더 본질적으로 '의미'에 맞게 작성하자는 태그
		<main>, <header>, <footer>, <aside>, <section>, <article>
		<b>가 아니라 <strong>
		폰트 크기가 아니라 h3
   SEO뭐지?
		검색엔진최적화.  Search Engine Optimization
		검색엔진 봇(bot)들이 정확하고 빠른 결과를 주기위해 웹문서들을 돌아다니면서 정보를 수집하는데
		HTML의 구조를 명확하고 시멘틱하게 짜서 검색결과에 우선순위로 노출되게 최적화하는 것.
   마크다운
		MarkDown.   마크업언어를 쓰면 콘텐츠와 메타데이터를 왔다갔다 하느라 (꺾쇠를 열고 닫고)
		    글을 쓰는 흐름이 계속 끊기게 되므로 마크업 문법을 매우 간단하게 바꾼 언어
			예) #(h1),  ###(h3),  - li,  [이름](주소) <a>
   
   
   ---CSS---
   뭐의 약자지?
		Cascading Style Sheet.  폭포수처럼 흐르는 디자인 표
		한놈에 여러가지 다른놈들이 동시에 서로다른 명령을 내릴수있다. 우선순위가 중요하다.
   우선순위
		추상적인것에서 구체적으로 갈수록 우선순위가 높아진다.
		태그 < 클래스 < 아이디
		외부스타일 < 내부스타일 < 인라인
		클래스 < 클래스안의 클래스 < 아이디안의 클래스
   왜쓰지?
		HTML의 본질은 정보이기 때문에 스타일에 관련된 것을 Html태그로 만들다가, 
		아예 스타일만 다루는 언어를 분리하기로 결정했고 '관심사의 분리'로 인해 유지보수가 편하게 된다.
   CSS 쓰는 3가지 방법
		위에 HTML정리할때 다룸
   요소에 CSS정의하는 문법
		선택자{
			속성 : 값;
			속성2 : 값2;
		}
   박스모델
		모든 HTML의 요소는 
		contents, padding, border, margin 의 박스형태로 구성되어있다는 약속
   선택자
		그냥태그 =   그냥 이름   예) h2{ color : red}
		클래스  =   이름앞에 .   예) .pokemon { color : blue}
		아이디  =    이름앞에 #   예)  #picka { color : yellow}
   디스플레이
		inline = 자기 영역만큼만 차지   대표 : span
		block = 한줄을 다차지         대표 : div
		inline-block = 두개 섞은것   대표 : button
   포지션
		html요소의 위치를 결정
		디폴트 : static.  웹의 일반적인 흐름
		상대적인 : relative   static을 기준으로 offset(top, right, bottom, left)만큼 이동
		절대적인 : absolute   static이 아닌 부모를 기준으로 offset만큼 이동

	프로그래밍언어는 컴파일 언어, 인터프리터 언어가 있다.
	컴파일 = 사람의 언어를 기계어로 바꾸는 작업.
			'A' = 10진수로 65 = 2진수로 01000001
	       "hi" = 010111010111011001(임시적인 예)
	처음부터 끝까지 번역을 먼저 돌리고 실행하는 언어 = 컴파일 언어      EX)C
	이번에 실행을 문장을 이번에 번역해서 실행하는 언어 = 인터프리터 언어   EX)자바스크립트, 파이썬
	인터프리트언어들은 대부분 다이나믹 타입트 랭귀지.  데이터타입이 컴파일때 결정되는게 아니고 실행때 결정.
