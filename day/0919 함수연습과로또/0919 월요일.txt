	<환경>
	오라클 홈페이지에서 JDK 11 다운받았음.
		Sun Microsystems(썬마이크로시스템즈) 에서 만들었는데 오라클이 인수했다.
		Open JDK가 완전 무료고, 사실은 우리가 어제 다운받은것은 기업에서 쓰려면 돈줘야되는거다.
		개인은 무료다. 상업용이 될려면? 아직 법적으로 다툴 여지가 있다.
		Mysql도 오라클이 인수했다.
	JDK : Java Development Kit(자바통합개발도구) -> 자바로 프로그램을 개발
	JRE : Java Runtime Environment(자바실행환경) -> 자바프로그램을 실행
	JVM : Java Virtual Machine(가바가상머신)
	
	<언어>
	- 컴파일언어 : 전체를 번역해놓고 읽는 언어 (다양한 에러를 실행하기전에 걸러줌)
	- 인터프리터 : 한줄씩 번역하고 읽는 언어
	- 자바는 섞은놈 : 개발자가 .java 개발하면, javac가 .class파일로 컴파일
			jvm이 .class파일을 한줄씩 인터프리터로 읽는다.
			
	- 객체지향언어의 대명사
	- 객체지향프로그래밍이라는것은 : 프로그램을 명령어들의 묶음이나 순서가 아니라
			객체들간의 관계로 보는 패러다임
	1. OS에 독립적이다.
	    = 프로그램을 만들어놓으면 실행환경이 윈도우든, 리눅스든, 안드로이드든, TV든 다 돌아간다.
	     JVM에 의해서.
	2. 비교적 배우기 쉽다.
		= 객체지향언어가 인간이 생각하는 방식이기 때문에
    3. 메모리관리를 직접 하지 않는다.
		= 가비지 컬렉터(garbage collector)가 알아서 해준다.
	4. 멀티쓰레드를 지원한다. 
		= 
	프로그램 = 디스크상에 존재
	프로세스 = 주메모리(램)에 로딩된 프로그램
	쓰레드 = 1개의 프로세스는 최소한 하나의 쓰레드를 가지고있다.
		작업을 수행하는 주체.    멀티쓰레드 = 작업을 수행하는 주체가 여러개다.
		예) 크롬에서 영상을 보면서, 키보드를 움직이면서, 마우스를 움직이면서, 파일은 다운받고있으면서
	멀티태스킹 : 태스크는 업무.
		동시에 여러 프로세스를 돌리는 것. 엑셀을 3개 띄워놨다.
		그런데 코어가 하나라도 멀티태스킹이 된다. 사람이 느끼지못할만큼 빨리 돌려막기를 통해서
		사람이 클릭한것이 바로 수행되기를 느끼기까지 0.2초가 필요하다면 
		컴퓨터는 영상을 실행하고 있는프로그램을 0.02초마다 돌리고,
		마우스 인벤트 인식을 0.02초마다 하고,
		다운로드를 0.02초다 하고,    ... 하나의 코어로 여러개를 돌릴수있다.
		진짜 멀티코어는 저 자체를 진짜로 병렬적으로 돌리는것.
	boat(배) 에서 최대한 달수있는 엔진이 4개고 현재 돌아가는 엔진이 3개면
	        멀티태스킹이 4개까지되는데 3개하는거고,
			하나의 엔진에 프로펠러가 3개가 돌면 그 프로세스는 멀티쓰레드 3개를 돌리고있는것
